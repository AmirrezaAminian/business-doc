<h1 dir="rtl">
طریقه استفاده در صفحات وب:
</h1>

<p dir="rtl">
کد زیر را در قسمت style یا فایل css وارد نمایید:
</p>


```css
@font-face {
  font-family: Dirooz;
  src: url('Dirooz.eot');
  src: url('Dirooz.eot?#iefix') format('embedded-opentype'),
       url('Dirooz.woff') format('woff'),
       url('Dirooz.ttf') format('truetype');
  font-weight: normal;
}
```
